如何导出浏览器书签？

Chrome

书签管理器 → 右上角菜单 → 导出书签

Firefox

书签 → 管理所有书签 → 导入和备份 → 导出书签为 HTML

Edge

收藏夹 → 管理收藏夹 → 更多选项 → 导出收藏夹
